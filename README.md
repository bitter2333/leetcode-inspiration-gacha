# 力扣灵感扭蛋机

一个帮助编程学习者随机选择力扣题目的互动式刷题工具。

页面核心是一台带动画的扭蛋机。用户可以选择题目类型和难度，点击抽题后播放球仓晃动、旋钮旋转、胶囊掉落与打开动画，最后展示随机抽中的题目。

## 功能

- 支持题目类型：算法、数据库、Shell、多线程、JavaScript、pandas
- 支持难度：简单、中等、困难
- 使用可更新的本地题库数据
- 结果卡片展示题目名称、分类、难度、简短提示和力扣中文站链接
- 响应式布局，适配桌面和移动端

## 使用

直接用浏览器打开 `index.html` 即可。

或者直接打开<https://leetcode.bitter2333.xyz>进行使用

## 更新题库

运行同步脚本，从力扣中文站拉取题目列表并生成 `data/problems.json` 和 `data/problems.js`：

```powershell
node scripts/sync-problems.mjs
```
